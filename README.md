# HTML 动态粒子爱心告白网页
### ❤️ 在线演示地址：https://Aventardo7777.github.io/love-heart-html-demo/
> 点击链接直接在线观看跳动爱心动画，无需本地打开文件

## 项目概述
纯前端HTML+CSS+JS实现的动态粒子爱心视觉动画，无后端、无第三方依赖，浏览器打开即可播放持续跳动的爱心特效，适合告白、创意视觉展示。

## 核心特性
- ✨ 粒子组合爱心动态呼吸跳动动画
- 🌸 柔和渐变粉色粒子视觉效果
- 📱 兼容电脑、手机浏览器直接访问
- 🧩 单文件整合全部代码，开箱即用
- 🌐 支持GitHub Pages一键部署分享链接

## 本地运行
1. 克隆仓库
git clone https://github.com/Aventardo7777/love-heart-html-demo.git
2. 双击打开 爱心代码.html 即可预览动画
